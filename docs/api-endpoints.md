# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Lyrics

- `GET /api/lyrics`
  - Song lyrics index
- `POST /api/lyrics`
- `GET /api/lyrics/:id`
- `PATCH /api/lyrics/:id`
- `DELETE /api/lyrics/:id`

### Annotations

- `GET /api/annotations`
- `POST /api/annotations`
- `GET /api/annotations/:id`
- `PATCH /api/annotations/:id`
- `DELETE /api/annotations/:id`
